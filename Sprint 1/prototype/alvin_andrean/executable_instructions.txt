HOW TO CREATE AN EXECUTABLE


PRE-REQUISITES:

- Make Sure that Python is installed
- alvinandrean.py file exists


STEP 1: Go to the directory of the python file (alvinandreean.py)
In the terminal type "cd path/to/file_name.py" (e.g., prototype\alvin_andrean)

STEP 2: Install the required package
In the terminal type "pip install pyinstaller"

STEP 3: Create the EXECUTABLE
In the terminal type "pyinstaller --onefile --windowed alvinandrean.py"

STEP 4: Find the executable file in the "dist" folder. The executable file name should be alvinandrean.exe
